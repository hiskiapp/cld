Caledi

Cassava Leaf Disease Classificator

Caledi is a cassava leaf disease classificator. It uses machine learning to classify cassava leaf disease. Using Tensorflow and Keras, Caledi can classify cassava leaf disease up to 85% accuracy.

Deepnote GitHub Repo

Classification

Caledi has 6 classes of cassava leaf disease. The classes are:

Healthy

Brown Leaf Spot

Brown Streak Disease

Green Might Damage

Mosaic Disease

Read Mite Damage
